Pengaturan Kirim

Pilih mode kirim. Mode Akun Telegram Premium memakai backend pribadi/MTProto supaya emoji premium akun kamu ikut terkirim. Versi ini dibuat simple tanpa Kunci Backend. Mode Bot tetap tersedia untuk RichMessageOri Bot API.

Emoji Premium Telegram

Tambahkan custom_emoji_id dari Telegram. Untuk mode Akun Premium, ID ini dikirim sebagai entity custom emoji lewat akun Telegram kamu. Tekan tombol 💎 di toolbar untuk menyisipkan emoji premium.

Data backend, token bot, dan emoji disimpan di browser/perangkat ini saja.
